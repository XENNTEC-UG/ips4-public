# XENNTEC IPS4 — Public Distribution

## X License (Free Download)

**X License** is a required dependency for all paid XENNTEC IPS4 apps. It handles license validation, status checks, and activation — completely free of charge.

### Download

Go to the [Releases](https://github.com/XENNTEC-UG/ips4-public/releases) tab and download the latest `.tar` file.

### Installation

1. Download the latest `.tar` from [Releases](https://github.com/XENNTEC-UG/ips4-public/releases)
2. In your IPS4 Admin Control Panel, go to **System > Applications > Install**
3. Upload the `.tar` file and follow the on-screen instructions

### Requirements

- **PHP 8.1** or higher
- **ionCube Loader** for PHP 8.1 ([download](https://www.ioncube.com/loaders.php))
- **IPS Community Suite 4.7** or higher
